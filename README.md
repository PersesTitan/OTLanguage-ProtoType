## 변수

### 변수 선언
</br> 변수는 [문자1글짜] + [변수타입] + [문자1글짜]로 선언하시면 됩니다.


> OTL          JAVA         
>ㅇㅈㅇ (정수) : int형 </br>
>ㅇㅉㅇ (정수) : long형 </br>
>ㅇㅂㅇ (블린) : boolean형 </br>
>ㅇㅁㅇ (문자) : String형 </br>
>ㅇㄱㅇ (글자) : char형 </br>
>ㅇㅅㅇ (실수) : float형 </br>
>ㅇㅆㅇ (실수) : double형 </br>
> ###### ㅇ 대신 다른 문자를 넣어서 변수이름 선언 가능함!!

### 변수 값 가져오기
</br> 변수값을 가져올려면 [:]+[변수이름]으로 선언하시면 됩니다.


```
예시 )
오ㅈ우 100


```
